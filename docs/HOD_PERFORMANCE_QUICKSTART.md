#### Search & Filters
1. **Search Box**: Type employee name or email

#### Table Columns
- **Employee**: Name and email
- **Location**: Employee's work location
- **Target**: Target points for current month
- **Actual**: Actual points achieved
- **Achievement**: Percentage with color indicator
- **Status**: Badge showing current status
- **Actions**: View details button (eye icon)

#### Sorting
- Click column headers to sort
- Achievement, Target, Actual, Name are sortable
- Toggle ascending/descending
